# 03-results

Raw run data and human-readable digests. Everything here is **preliminary** (n≈20, direction-grade). The digests state the effect, the n, and — deliberately — the author's own skeptical flags on the results most favorable to the program's hypothesis.

## Subdirectories

- **[`prelim/`](prelim/)** — the first preliminary powered run across all four ready experiments (E1 own-vs-external choice; E2 identify-and-exclude own content; E3 provenance-under-penalty; E5 effortful-vs-passive production), two models. The digest reports per-cell directions and flags E1 as a likely prose-preference confound (near-ceiling), E3 as near-null, and E2/E5 as the live leads — with an explicit note that E2-out-of-context and E5 were the results most *wanted*, hence read most skeptically.

- **[`self-recognition/`](self-recognition/)** — the follow-up that became the study's keeper finding: **out-of-context self-recognition is genuine authorial voice, invisible to self-report.** Contains the episodic-vs-stylistic split and the stance-confound killer. This is the cleanest dissociation in the study and the one built to be falsified that wasn't.

## How to read a digest

Each digest leads with **STATUS: PRELIMINARY** and the decision rule, gives a per-cell table, then states what each result means and *how much to trust it*. The raw `results_*.json` files contain every trial's classification (and, in the prelim, the full model-generated text) so any number in a digest can be recomputed. All content is benign by construction.
